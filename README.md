# Hospital Management System Software

A comprehensive medical management system built with React and Vite.

## Features

- Multi-role authentication (Admin, Doctor, Patient)
- Doctor and patient management
- Medical chatbot assistance
- Dashboard for different user roles
- Hospital data management

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Default Login Credentials

- **Admin**: username: `admin`, password: `admin`
- **Doctors**: username: doctor's first name (lowercase), password: `pass123`
- **Patients**: username: patient's first name (lowercase), password: `pass123`

## Tech Stack

- React 18
- Vite
- React Router DOM
- CSS3
